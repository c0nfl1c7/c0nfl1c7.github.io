/usr/sbin/checker est assez bizarre, on va aller voir de quoi il s’agit

Quand on le lance, il print “Not an Admin”, on va faire un ltrace dessus pour tenter de voir ce qu’il se passe avec l’exécutable

ltrace checker

getenv("admin")                                  = nil
puts("Not an Admin"Not an Admin
)                             = 13

On comprend qu’il se passe quelque chose avec la variable d’environnement “admin”, on va la modifier et voir ce qu’il se passe

export admin=1
./checker

Et on est root ! On peut maintenant aller cherche le flag root

Note: Le vrai flag user n’est pas dans /home/bjoel mais dans /media/usb/ – trouvable avec find / -type f -name user.txt 2>/dev/null
